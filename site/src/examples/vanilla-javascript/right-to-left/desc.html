In this demo, we add dir="rtl" to the library's container so the layout flows right-to-left. Phone numbers and dial codes are still displayed left-to-right, as that’s the standard way to write telephone numbers. We also import the Arabic locale and pass it via the i18n option, set countryNameLocale to "ar" (Arabic), and initialise the library with initialCountry set to Egypt.

Note: the toast that appears when strictMode rejects input isn't part of the library — it's wired up via the strict:reject event (whose docs include a full example).
