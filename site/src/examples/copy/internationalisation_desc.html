There are two aspects to the internationalisation of this plugin. The first is the country names, which are translated automatically using the native Intl.DisplayNames - just specify the locale you want using the countryNameLocale option. The second is the user interface strings (including the country search placeholder, search empty state, and various other strings for screen readers). We provide translations for these in over 40 languages - simply import the language module you need and pass it to the plugin using the i18n option, as in the example below. Alternatively, you can specify your own translations - for more details, see the i18n option. View all of the provided translations in action in the Playground.
